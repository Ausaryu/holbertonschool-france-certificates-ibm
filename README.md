# Holberton School France - IBM Certificates

This repository tracks the IBM badges earned during the Holberton School France program. Each trimester folder keeps PDF copies of the certificates plus links to the official issuers for verification.

## Repository layout
- certificates-trimester-1/ - Trimester 1 badges (Agile Explorer, Open Source and Linux). See certificates-trimester-1/README.md for details.
- certificates-trimester-2/ - Trimester 2 certificates (Project Management 1, SQL). See certificates-trimester-2/README.md for details.
- certificates-trimester-3/ - Trimester 3 certificates (Cybersecurity Fundamentals, Cloud Computing Fundamentals). See certificates-trimester-3/README.md for details.

## How to use
- Open the trimester README to find badge names, local PDF files, and the official verification links.
- Use the Credly links to confirm authenticity; the PDFs are provided for quick offline reference.
- Add future trimesters as new folders with their own README and badge links.
